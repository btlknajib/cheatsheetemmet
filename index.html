Emmet c'est quoi ?

Ce petit plugin va permettre simplement et efficacement d’accélérer votre rythme de développement. Si j'annonce que je vais créer un tableau HTML contenant 8 lignes de 5 cellules chacune, contenant elles aussi 8 liens chacune. Et si, j’ajoute que tout ce contenu HTML, je vais le générer grâce à une seule ligne de code? Vous allez rire, mais pas si je fais tout ça avec Emmet ...

Syntaxe

  • Ajouter un enfant ou Child :

    nav>ul>li donne : <nav>
        <ul>
           <li></li>
        </ul>
    </nav>

  • Ajouter un élément frère ou Sibling :

    div+ul+li donne : <nav>
    <ul>
    <li>

  • Multiplier les éléments :

    ul>li*5 donne :
    <ul>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
        <li></li>
    </ul>

  • Commencer la numérotation à X nombre :

    ul>li.item$@3*5 donne :
    <ul>
        <li> class="item3"></li>
        <li> class="item4"></li>
        <li> class="item5"></li>
        <li> class="item6"></li>
        <li> class="item7"></li>
    </ul>

  • Ordre décroissant :

    ul>li.item$@-*5 donne :
    <ul>
        <li> class="item5"></li>
        <li> class="item4"></li>
        <li> class="item3"></li>
        <li> class="item2"></li>
        <li> class="item1"></li>
    </ul>

  • Remonter d'un élément enfant :

    <div>p>em{hello}^a donne :
      <div>
        <p> <em> hello </em> </p>
      <a href=""> /a>
    </div>

  • Ajouter un attribut :

    img[src="URL" alt="ALT"] donne :
    <img src="URL" alt="ALT">

  • Ajouter une classe ou un ID :

    ul.wrapper donne :
    <ul> class="wrapper"</ul>

    ul#item0 donne :
    <ul> id="item0"</ul>
